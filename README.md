Project Overview:

The Event Management System is an integrated digital platform designed to improve event organization by centralizing ticketing, registration, payments, venue navigation, session selection, networking, and sponsor advertising.
This MVP is developed as part of the final project for the Project Management course – Group 11.


Core Features (MVP):

Ticketing & Registration
Secure Payment Integration (Stripe/PayPal Sandbox)
Session Selection Assistant
Venue Map & Seat Selection
Sponsor Ad Module
Attendee Networking
Organizer Dashboard (Basic Analytics)
